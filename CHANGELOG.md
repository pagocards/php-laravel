# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-06

### Added
- Initial release of Pagocards PHP SDK
- Client class for API communication
- Methods for creating Mastercard and Visa cards
- Methods for funding cards
- Methods for retrieving card details and listing cards
- Methods for blocking/unblocking cards
- Methods for checking wallet balance
- Laravel Service Provider for easy integration
- Facade for convenient usage in Laravel
- Configuration file support
- Comprehensive test suite
- Full documentation and README
- MIT License

### Features
- PSR-4 autoloading
- Support for Laravel 8, 9, 10, and 11
- PHP 8.1+ support
- Guzzle HTTP client integration
- Error handling with detailed exceptions
- Chainable method calls
